# Reports Directory

This folder contains experiment outputs intended for sharing:

- `experiment_suite_results.json`: raw structured run-level metrics.
- `experiment_suite_results.csv`: flat table for spreadsheet analysis.
- `experiment_report_2026-04-18.md`: human-readable summary.
- `rl_trading_experiment_report_2026-04-18.pdf`: final PDF report.

Generated by:

```powershell
python scripts/run_experiment_suite.py --project-root .
python scripts/build_report_markdown.py --summary-json docs/reports/experiment_suite_results.json --output-md docs/reports/experiment_report_2026-04-18.md
python scripts/build_pdf_report.py --summary-json docs/reports/experiment_suite_results.json --output-pdf docs/reports/rl_trading_experiment_report_2026-04-18.pdf
```
